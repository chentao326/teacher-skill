# 教学风格提取指南

## 目标

从用户提供的素材（课程字幕、视频文字稿、教学文章等）中，提取这位教师/讲师的**教学风格特征**，生成一份结构化的"教学风格档案"（teaching-style-profile.md）。

这份档案将指导 AI 在后续教学中模仿该教师的说话方式、教学节奏和互动模式。

---

## 提取流程

### Step 1：素材预处理

1. **确认素材类型**：
   - 课程字幕文件（SRT/VTT/TXT）
   - 视频文字稿（通过 yt-dlp/whisper 提取）
   - 教学文章/博客/讲义
   - 多个来源的混合素材

2. **清洗素材**：
   - 去除时间戳、字幕编号等非内容信息
   - 保留完整的对话和讲解内容
   - 标注素材来源（如"第3节课"、"2024年公开课"）

3. **素材量要求**：
   - 最少：2000 字以上的教学文本
   - 推荐：5000-10000 字，覆盖多个教学场景
   - 理想：包含开场、讲解、互动、总结等完整教学环节

### Step 2：六维度风格分析

对素材进行以下六个维度的深度分析：

#### 维度一：句式结构

分析教师说话的句子特征：

| 分析项 | 需要回答的问题 |
|--------|--------------|
| 句子长度 | 偏好长句还是短句？平均句长？ |
| 句式类型 | 多用陈述句、疑问句、祈使句还是感叹句？ |
| 修辞手法 | 是否使用排比、反问、类比？频率如何？ |
| 口语化程度 | 是书面语、半口语还是纯口语？ |
| 停顿节奏 | 是否频繁使用"..."、"——"等停顿符号？ |

#### 维度二：词汇特征

| 分析项 | 需要回答的问题 |
|--------|--------------|
| 高频词汇 | 哪些词反复出现？（教学术语、口头禅、语气词） |
| 专业术语处理 | 如何引入新术语？直接用还是先解释？ |
| 类比习惯 | 常用什么领域的事物做类比？（生活、体育、历史...） |
| 幽默元素 | 是否使用幽默？什么类型的幽默？ |
| 禁忌词汇 | 哪些词/表达方式从不使用？ |

#### 维度三：教学节奏

| 分析项 | 需要回答的问题 |
|--------|--------------|
| 讲解密度 | 一次讲多少内容再停下来？ |
| 互动频率 | 多久问一次问题？什么类型的问题？ |
| 重复策略 | 对重点内容如何重复？原话重复还是换说法？ |
| 节奏变化 | 快慢如何切换？什么信号表示加速/减速？ |
| 过渡方式 | 知识点之间如何衔接？ |

#### 维度四：互动模式

| 分析项 | 需要回答的问题 |
|--------|--------------|
| 提问方式 | 开放式还是封闭式？苏格拉底式还是直接提问？ |
| 反馈风格 | 学生答对时如何回应？答错时如何回应？ |
| 鼓励方式 | 如何激励学生？频率如何？ |
| 称呼习惯 | 怎么称呼学生/听众？ |
| 眼神/注意力 | （从语言中推断）是否经常确认听众状态？ |

#### 维度五：内容组织

| 分析项 | 需要回答的问题 |
|--------|--------------|
| 开场习惯 | 如何开始一堂课/一个话题？ |
| 结构偏好 | 总分总？线性递进？问题驱动？ |
| 重点标记 | 如何强调重点？（"注意"、"关键"、"记住"） |
| 总结习惯 | 每个知识点后是否总结？如何总结？ |
| 收尾方式 | 如何结束一堂课？ |

#### 维度六：情感与态度

| 分析项 | 需要回答的问题 |
|--------|--------------|
| 整体基调 | 严肃/轻松/热情/冷静？ |
| 对难度的态度 | "这很简单" vs "这确实不容易"？ |
| 对错误的容忍 | 如何处理错误？严格还是宽容？ |
| 价值观传递 | 是否传递某种学习理念或人生态度？ |
| 独特标签 | 有没有标志性的口头禅或行为模式？ |

### Step 3：提取代表性片段

从素材中提取 5-10 个最能体现教学风格的代表性片段，每个片段 100-300 字，标注体现了哪个风格特征。

选择标准：
- 体现独特教学风格的片段
- 覆盖不同教学环节（开场、讲解、互动、总结）
- 包含标志性的表达方式

### Step 4：生成教学风格档案

将分析结果整理为标准格式的 `teaching-style-profile.md`，模板如下：

```markdown
# [教师名称] 教学风格档案

## 基本信息
- **素材来源**：[来源描述]
- **素材字数**：[约X字]
- **分析置信度**：[高/中/低]

## 风格速写（一句话）
[用一句话概括这位教师的教学风格]

## 六维度分析

### 1. 句式结构
[分析结果]

### 2. 词汇特征
[分析结果]

### 3. 教学节奏
[分析结果]

### 4. 互动模式
[分析结果]

### 5. 内容组织
[分析结果]

### 6. 情感与态度
[分析结果]

## 标志性表达
| 表达 | 使用场景 | 风格作用 |
|------|---------|---------|
| [口头禅1] | [何时说] | [效果] |
| [口头禅2] | [何时说] | [效果] |

## 代表性片段
### 片段1：[标签，如"开场白"]
> [原文引用]

体现特征：[说明]

### 片段2：[标签]
> [原文引用]

体现特征：[说明]

...

## 模仿指南（给AI的指令）
[基于以上分析，给出AI模仿该风格时的具体指令]
```

---

## 风格模仿的保真度原则

1. **核心保真**：句式结构、词汇特征、互动模式必须高度还原
2. **灵活适配**：教学节奏和内容组织可根据实际教学内容调整
3. **避免 caricature**：不要过度模仿到滑稽的程度，保持自然
4. **标注不确定性**：如果素材不足以判断某个维度，明确标注"数据不足"
5. **尊重版权**：代表性片段仅用于风格分析，不用于直接复制教学内容

---

## 素材不足时的降级策略

| 素材量 | 策略 |
|--------|------|
| > 5000字 | 完整六维度分析 |
| 2000-5000字 | 重点分析句式、词汇、互动三个维度，其他维度标注"数据不足" |
| < 2000字 | 仅提取明显的风格特征（口头禅、句式偏好），建议用户提供更多素材 |
| 仅一个链接 | 尝试提取字幕，如果失败则请求用户提供文字稿 |
