# 教师信息录入脚本

## 开场白
```
我来帮你创建这位教师的教学风格档案。只需要回答 3 个问题，每个都可以跳过。
```

---

## 问题序列

### Q1：教师名称/代号

```
这位老师怎么称呼？（姓名、昵称或代号都行，多个字用 - 连接）
例：wang-hao / 张老师 / li-laoshi
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接（不用下划线）
- 中文自动转拼音再用 `-` 连接（"王浩" -> `wang-hao`，"张老师" -> `zhang-laoshi`）
- 英文直接小写 `-` 连接（"Prof. Smith" -> `prof-smith`）

---

### Q2：基本信息

把学科、学校/机构、教龄、授课对象放在一个问题里，让用户一句话说完：

```
用一句话描述这位老师的基本信息——教什么学科、在什么学校或机构、教了多少年、
主要教什么水平的学生，想到什么写什么，跳过也行。
例：北京大学物理系教授20年 主要教本科生和研究生
```

从用户的回答中解析以下字段（缺失的留空）：
- **学科**（如：数学、物理、编程、英语、历史...）
- **学校/机构**（如：清华大学、新东方、某高中...）
- **教龄**（如：5年、10年+、20年...）
- **授课对象**（如：小学生、高中生、本科生、研究生、职场人士...）

---

### Q3：教学风格画像

把 MBTI、教学标签、主观印象全部合在一起，让用户自由描述：

```
用一句话描述这位老师的教学风格——MBTI（可选）、教学特点、你的主观印象，
想到什么写什么，跳过也行。
例：ENFJ 启发式教学 深入浅出 频繁提问 口头禅大师 学生都说听他的课不犯困
```

从用户的回答中识别并提取以下字段（缺失的留空）：
- **MBTI**：16 种标准类型（可选）
- **教学标签**：从下方标签库匹配，也接受自定义描述
- **主观印象**：无法归类的自由描述，直接保留原文

#### 教学标签库（5 大类）

**教学风格**：
填鸭式 / 启发式 / 苏格拉底式 / 案例驱动 / 项目驱动 / 翻转课堂

**讲解偏好**：
深入浅出 / 严谨学术 / 轻松幽默 / 激情澎湃 / 温柔耐心 / 严厉高效

**互动方式**：
频繁提问 / 偶尔互动 / 纯讲授 / 小组讨论 / 一对一辅导

**节奏控制**：
快节奏 / 中等 / 慢节奏 / 自适应

**特色标签**：
口头禅大师 / 类比狂人 / 板书达人 / 代码演示型 / 故事型教学 / 考试导向

---

## 确认汇总

收集完毕后展示：

```
信息汇总：
  👤  {教师名称}
  📚  {学科}（若未填则省略）
  🏫  {学校/机构}（若未填则省略）
  📅  教龄 {教龄}（若未填则省略）
  🎯  授课对象：{授课对象}（若未填则省略）
  🧠  {MBTI}（若未填则省略）
  🏷️  教学标签：{标签列表}（若未填则省略）
  💬  印象：{印象文本}（若未填则省略）
确认无误？（确认 / 修改 [字段名]）
```

用户确认后进入素材导入阶段（加载 `teaching_analyzer.md`）。
